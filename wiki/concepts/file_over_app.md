# File Over App

The philosophy that data should live in universal, interoperable file formats (markdown, images) rather than being locked inside a specific application's proprietary system.

## Key Points

- Data in universal formats is interoperable: any tool, CLI, or agent can operate on it
- You can use Obsidian to view, vibe-code your own viewer, or apply Unix tools directly
- Any data can be imported as input; any interface can render it as output
- You stay in control of your information — not locked into a particular AI provider
