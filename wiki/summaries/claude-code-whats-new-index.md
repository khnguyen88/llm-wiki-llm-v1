# Claude Code Whats New Index

The What's New index is a weekly dev digest highlighting the Claude Code features most likely to change how developers work. Each entry links to a detailed weekly page with runnable code, demos, and links to full documentation. Bug fixes and minor improvements are covered in the separate changelog. ^[raw/document/claude code/claude-code-123-whats-new-index-2026-04-29.md]

## Key Points

- **Week 17** (v2.1.114-119, Apr 20-24): `/ultrareview` opens as a public research preview with a fleet of bug-hunting agents in the cloud; session recap shows what happened while a terminal was unfocused; custom themes allow building and shipping color palettes from `/theme` or a plugin; Claude Code on the web receives a redesign with a sessions sidebar and drag-and-drop layout ^[raw/document/claude code/claude-code-123-whats-new-index-2026-04-29.md]
- **Week 16** (v2.1.105-113, Apr 13-17): Claude Opus 4.7 becomes the new default on Max and Team Premium with an `xhigh` effort level recommended for most coding work and an interactive `/effort` slider; Routines on the web fire templated cloud agents from a schedule, GitHub event, or API call; `/usage` shows what drives limits; `/ultrareview` runs parallel multi-agent review in the cloud; the CLI moves to native binaries ^[raw/document/claude code/claude-code-123-whats-new-index-2026-04-29.md]
- **Week 15** (v2.1.92-101, Apr 6-10): Ultraplan enters early preview for drafting plans in the cloud from the CLI, with review and commenting in a web editor, then remote or local execution; the Monitor tool streams background events into the conversation; `/loop` self-paces when the interval is omitted; `/team-onboarding` packages setup into a replayable guide; `/autofix-pr` enables PR auto-fix from the terminal ^[raw/document/claude code/claude-code-123-whats-new-index-2026-04-29.md]
- **Week 14** (v2.1.86-91, Mar 30-Apr 3): Computer use comes to the CLI in research preview for closing the loop on GUI-only tasks; `/powerup` provides interactive lessons; flicker-free alt-screen rendering; per-tool MCP result-size override up to 500K; plugin executables on the Bash tool's PATH ^[raw/document/claude code/claude-code-123-whats-new-index-2026-04-29.md]
- **Week 13** (v2.1.83-85, Mar 23-27): Auto mode lands in research preview as a classifier-based middle ground between approving everything and `--dangerously-skip-permissions`; computer use in the Desktop app; PR auto-fix on Web; transcript search with `/`; a native PowerShell tool for Windows; conditional `if` hooks ^[raw/document/claude code/claude-code-123-whats-new-index-2026-04-29.md]
- The complete documentation index is available at `https://code.claude.com/docs/llms.txt` for discovering all available pages ^[raw/document/claude code/claude-code-123-whats-new-index-2026-04-29.md]
- Bug fixes and minor improvements beyond the weekly highlights are tracked in the changelog at `https://code.claude.com/docs/en/changelog` ^[raw/document/claude code/claude-code-123-whats-new-index-2026-04-29.md]
