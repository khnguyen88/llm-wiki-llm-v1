# OpenRouter Guides: Free Model Variant

## Key Points

- The `:free` variant suffix allows access to free versions of models on OpenRouter ^[raw/document/openrouter/openrouter-020-guides-routing-model-variants-free-2026-04-29.md]
- Append `:free` to any model ID (e.g., `"meta-llama/llama-3.2-3b-instruct:free"`) to use the free variant ^[raw/document/openrouter/openrouter-020-guides-routing-model-variants-free-2026-04-29.md]
- Free variants provide model access without cost but may have different rate limits or availability compared to paid versions ^[raw/document/openrouter/openrouter-020-guides-routing-model-variants-free-2026-04-29.md]
- A separate Free Models Router is available in the Chat Playground for zero-cost inference ^[raw/document/openrouter/openrouter-020-guides-routing-model-variants-free-2026-04-29.md]
