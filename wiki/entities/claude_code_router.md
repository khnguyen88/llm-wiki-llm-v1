# Claude Code Router

Open-source project by musistudio on GitHub (`musistudio/claude-code-router`). NPM package: `@musistudio/claude-code-router`. Routes Claude Code requests to arbitrary LLM providers via a local proxy server.

## Key Facts

- Language: TypeScript/JavaScript
- Runtime: Node.js >= 18
- Install: `npm install -g @musistudio/claude-code-router`
- Default port: 3456
- Core library: `@musistudio/llms` (standalone universal LLM API transformation library)
- Docker image: `musistudio/claude-code-router:latest`
- Sponsored by Z.ai (GLM Coding Plan)
