# Claude Console

Anthropic's API management platform that provides API-based billing and user management for organizations using Claude Code. Console authentication is an alternative to Claude for Teams/Enterprise for teams that prefer pay-as-you-go billing. ^[001a-raw/document/claude code/claude-code-034-authentication-2026-04-29.md]

## Key Facts

- Supports API-based billing as an alternative to per-seat Claude for Teams/Enterprise plans ^[001a-raw/document/claude code/claude-code-034-authentication-2026-04-29.md]
- Users are added through bulk invite from Settings > Members > Invite or via SSO setup ^[001a-raw/document/claude code/claude-code-034-authentication-2026-04-29.md]
- Two user roles: Claude Code role (can only create Claude Code API keys) and Developer role (can create any kind of API key) ^[001a-raw/document/claude code/claude-code-034-authentication-2026-04-29.md]
- Each invited user must accept the Console invite, check system requirements, install Claude Code, and log in with Console account credentials ^[001a-raw/document/claude code/claude-code-034-authentication-2026-04-29.md]
