# Claude Code Whats New

A weekly digest of notable Claude Code features, with each entry covering version ranges and brief descriptions. The source covers weeks 13-17 (March 23 - April 24, 2026), spanning versions v2.1.83 through v2.1.119. ^[001a-raw/document/claude code/claude-code-117-whats-new-2026-04-29.md]

## Key Points

- **Week 17** (v2.1.114-119, Apr 20-24): `/ultrareview` opens as a public research preview; session recap shows what happened while a terminal was unfocused; custom themes let users build and ship color palettes from `/theme` or a plugin; Claude Code on the web gets a redesign with a sessions sidebar and drag-and-drop layout ^[001a-raw/document/claude code/claude-code-117-whats-new-2026-04-29.md]
- **Week 16** (v2.1.105-113, Apr 13-17): Claude Opus 4.7 becomes the default model on Max and Team Premium with a new `xhigh` effort level recommended for most coding work; an interactive `/effort` slider dials it in; Routines on Claude Code on the web fire templated cloud agents from a schedule, GitHub event, or API call; `/usage` shows what is driving your limits; the CLI moves to native binaries ^[001a-raw/document/claude code/claude-code-117-whats-new-2026-04-29.md]
- **Week 15** (v2.1.92-101, Apr 6-10): Ultraplan enters early preview for drafting plans in the cloud from the CLI; the Monitor tool streams background events into the conversation; `/loop` self-paces when the interval is omitted; `/team-onboarding` packages your setup into a replayable guide; `/autofix-pr` turns on PR auto-fix from the terminal ^[001a-raw/document/claude code/claude-code-117-whats-new-2026-04-29.md]
- **Week 14** (v2.1.86-91, Mar 30-Apr 3): Computer use comes to the CLI in research preview; `/powerup` provides interactive lessons; flicker-free alt-screen rendering; a per-tool MCP result-size override up to 500K; plugin executables on the Bash tool's PATH ^[001a-raw/document/claude code/claude-code-117-whats-new-2026-04-29.md]
- **Week 13** (v2.1.83-85, Mar 23-27): Auto mode lands in research preview as a middle ground between approving everything and `--dangerously-skip-permissions`; computer use in the Desktop app; PR auto-fix on Web; transcript search with `/`; a native PowerShell tool for Windows; conditional `if` hooks ^[001a-raw/document/claude code/claude-code-117-whats-new-2026-04-29.md]
